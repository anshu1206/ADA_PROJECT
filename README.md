# ADA Project - Instagram Collage Maker

An intuitive and powerful web application designed to help you create stunning collages for Instagram. With drag-and-drop functionality, multiple layout options, and built-in optimization tools, creating perfectly sized social media content has never been easier.

## Features

- **Multiple Canvas Sizes**: Choose from standard social media dimensions:
  - 📱 Portrait (1080x1350)
  - ⬛ Square (1080x1080)
  - 🖼️ Landscape (1080x566)
  - 📲 Story (1080x1920)
  - 🎬 YouTube Thumbnail (1280x720)
- **Flexible Layouts**: Automatically arrange your photos using pre-defined layouts:
  - Horizontal (Side by Side)
  - Vertical (Stacked)
  - Grid
  - Mixed
  - Divide & Conquer (Randomized split layouts)
- **Drag & Drop Interface**: Easily upload images by dropping them into the app, and reorder images and slides with drag-and-drop.
- **Image Manipulation**: Zoom in, zoom out, and reposition images within their frames for the perfect crop.
- **Multiple Slides**: Create carousel posts by adding multiple slides and managing them all in one place.
- **Export Options**: 
  - Download individual slides with granular quality control.
  - Download all slides as a ZIP file.
  - Auto-optimize for Instagram to keep file sizes under 1MB.

## Technologies Used

- **HTML5**
- **CSS3** (Vanilla)
- **JavaScript** (Vanilla)
- **HTML Canvas API**

## How to Use

1. Open `index.html` in your web browser.
2. Select your desired **Canvas Size** from the left panel.
3. **Upload Images** by clicking the dropzone or dragging and dropping files.
4. Select a **Layout** from the right panel to arrange your images.
5. Hover over an image frame to reveal controls to zoom or drag to reposition the photo.
6. Use the slide navigation at the bottom to create and manage multiple slides.
7. Click the **Download** button or the **Auto-Optimize for Instagram** button to export your collage.

## Project Structure

- `index.html`: The main structural layout of the application.
- `style.css`: Contains all the styling for the user interface.
- `script.js`: Contains the logic for file uploads, canvas rendering, layouts, and exporting.

## Running Locally

Since the application uses standard web technologies, you can simply open the `index.html` file in any modern web browser to start using it. No build tools or local servers are required!
